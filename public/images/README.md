# Imagens dos Participantes

Coloque as fotos dos participantes nesta pasta com os seguintes nomes:

- `rafael.jpg` - Foto do Rafael Malaguti
- `lincoln.jpg` - Foto do Lincoln Roncato
- `natalia.jpg` - Foto da Natalia Souza (quando disponível)

As imagens devem estar em formato JPG, PNG ou WEBP e serão exibidas automaticamente na página de Participantes.
